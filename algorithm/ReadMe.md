常用的数据结构
---------------
>-数组    (array)  
>-栈      (stack)  
>-队列    (queue)  
>-链表    (linked list)  
>-树      (tree)  
>-图      (graph)  
>-堆      (heap)  
>-散列表  (hash)  

#####线性表(数组，栈、队列等)：
特点：  
1、1-2-3-4-...-n(n>=0)  
2、数据类型相同  
3、长度相同  

#####树：
特点：
1、有且只有一个节点没有直接前驱，就是根节点。  
2、除根节点，每一个节点都有且只有一个直接前驱。  
3、每一个节点都可以有任意多个直接后继。
